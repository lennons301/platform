# Testing Standard

Every project has automated tests that can be run locally.

## Requirements

1. A test command exists and is documented in CLAUDE.md (e.g., `npm test`, `npx vitest`).
2. Tests run against local services (not remote/production).
3. CI runs tests on every PR.

## Guidance

The specific testing framework and strategy is a project-level decision. This standard only requires that tests exist and are runnable.
